Important FAQ ?

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is the programming interface. DOM html and xml files look like tree structure. It represents the Node collection, the HTML collection. We can dynamically manipulate an HTML file using event handlers.

2. How do you select an element from the DOM?

We can use some event handlers to select an element form the DOM.

Select an single element method.

  • document.getElementById()
  • document.querySelector()

Select more then one elements method.

  • document.getElementsByClassName()
  • document.querySelectorAll()
3. What is event delegation in the context of the DOM, and why is it useful?

The event delegation technique in js.The event delegation technique in JavaScript allows us to target a parent element and use it to manage events for its child elements.No need to write different functions for childs so no code repeats Less memory consumption so site loading fast .

4. How do you manipulate an element's attributes and styles using the DOM?

I will select nodes or elements from DOM model using selector method. Then, based on the type of element, we can add different event handlers(get attributes, set attributes, classlist ,.style) or manipulate the selected element by adding style properties by associating events with different functions.
